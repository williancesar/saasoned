# Shared vocabulary (canonical — defined once)

Authoritative definitions for the whole suite. Siblings **use** these terms
exactly and link here; they do **not** restate or fork these definitions. Each
term: one-line definition · source · which sibling owns the deep mechanics.

| Term | Definition (one line) | Source | Owned by |
|---|---|---|---|
| **Distribution channel** | A repeatable path through which the right users reach the product. | *Zero to One*; *Traction* | growth-traction-channels |
| **Traction (vs. product)** | Quantitative evidence the business is taking off; the *separate track* from building the thing. | *Traction* | growth-traction-channels |
| **50% rule** | Spend ~half of effort on traction, ~half on product, concurrently. | *Traction* | hub (worldview) |
| **Critical Path** | The shortest sequence of milestones to the traction goal; do only product work that serves it. | *Traction* | growth-experiments |
| **Monopoly / niche-monopoly** | Owning a market small enough to dominate, then expanding from strength. | *Zero to One* | growth-positioning |
| **Power law (of returns)** | A few bets/outcomes dominate all others; concentrate, don't spread. | *Zero to One* | hub (worldview) |
| **Distribution power law** | Among channels, typically ONE dominates; find and focus it. | *Zero to One*; *Traction* | growth-traction-channels |
| **Dead zone** | The mid-CAC gap where a deal is too big for cheap viral/ads yet too small for direct sales — nobody can afford to sell it. | *Zero to One* | growth-traction-channels |
| **CAC** | Customer acquisition cost: fully-loaded cost to acquire one paying customer via a channel. | *Zero to One* | growth-traction-channels |
| **CLV / LTV** | Customer lifetime value; viable channel needs CLV > CAC with margin. | *Zero to One* | growth-experiments |
| **Bullseye** | Brainstorm channels → rank into rings → cheap tests → focus the one with signal. | *Traction* | growth-traction-channels |
| **Inner / middle / outer ring** | Bullseye ranking: most-promising (test now) / possible / long-shot. | *Traction* | growth-traction-channels |
| **Traction signal** | A channel test result good enough to justify focusing/scaling it (vs. noise). | *Traction* | growth-experiments |
| **Technology Adoption Life Cycle** | Innovators → early adopters → early majority → late majority → laggards. | *Crossing the Chasm* | growth-positioning |
| **The chasm** | The gap between early adopters and the pragmatic early majority where products stall. | *Crossing the Chasm* | growth-positioning |
| **Beachhead** | The single niche you attack first to cross the chasm. | *Crossing the Chasm* | growth-positioning |
| **Whole product** | Everything (integrations, support, proof) needed for the target segment to fully adopt, not just the core. | *Crossing the Chasm* | growth-positioning |
| **Positioning** | The deliberate context (category, alternatives, attributes, value, best-fit customer) that makes value obvious. | *Obviously Awesome* | growth-positioning |
| **5(+1) components** | Competitive alternatives · unique attributes · value & proof · best-fit customers · market category · (+ trend). | *Obviously Awesome* | growth-positioning |
| **Atomic network** | The smallest network that is stable and useful on its own; growth = adding more of these. | *The Cold Start Problem* | growth-network-effects |
| **Hard side** | The minority of users who do the disproportionate work a network depends on; hardest to acquire/retain. | *The Cold Start Problem* | growth-network-effects |
| **Cold Start → … → Moat** | The arc: Cold Start → Tipping Point → Escape Velocity → Ceiling → Moat. | *The Cold Start Problem* | growth-network-effects |
| **Anti-network effect** | When more users *degrade* value (spam, congestion, low quality) — a network can shrink. | *The Cold Start Problem* | growth-network-effects |
| **Hook Model** | Trigger → Action → Variable Reward → Investment, repeated until habitual. | *Hooked* | growth-habit-loops |
| **Fogg Behavior Model** | Behavior = Motivation × Ability × Trigger, all present at once. | *Hooked* (Fogg) | growth-habit-loops |
| **Habit Zone** | Behaviors of sufficient frequency *and* perceived utility become habits. | *Hooked* | growth-habit-loops |
| **Manipulation Matrix** | Ethics gate: would the maker use it / does it materially improve the user's life. | *Hooked* | growth-habit-loops |
| **The Mom Test** | Talk about their life and past behavior, not your idea; opinions/compliments/hypotheticals are bad data. | *The Mom Test* | growth-customer-discovery |
| **Commitment / advancement** | Real interest is shown by spending a currency (time, reputation, money) and moving to a next step. | *The Mom Test* | growth-customer-discovery |
| **Audience-first / Audience–Product Fit** | Find and embed in an audience before/while building; fit the product to a known audience. | *The Embedded Entrepreneur* | growth-customer-discovery |
| **North Star / one-metric-that-matters** | The single metric capturing delivered value that the team optimizes now. | *Traction* / *Lean Analytics* lineage | growth-experiments |
| **Vanity metric** | A number that rises without proving value or informing a decision. | growth canon | growth-experiments |
| **Premature scaling** | Spending to scale a channel/network/product before its success signal exists. | *Traction*; *Cold Start Problem* | growth-experiments |

**Attribution rule (applies to all siblings).** Always name the author/book for
a framework. Paraphrase faithfully. Never fabricate verbatim quotes, page
numbers, or statistics; if unsure of a figure, state the principle without a
number.
